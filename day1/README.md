# Assignment 1: Rule-Based AI Agent

## Objective
Build a simple AI agent using rule-based logic.

## Architecture
The agent follows a simple pipeline:

Input → Intent Detection → Action Execution → Output

## Components

- agent.py
  - Handles user input
  - Determines intent using keyword matching

- actions.py
  - Contains functions for:
    - Greeting
    - Date retrieval
    - Calculation

## Features
- Greeting detection
- Date retrieval
- Arithmetic calculations
- Unknown input handling

## Example Usage
User: calculate 2 + 3  
Output: Result: 5

## Learning Outcome
- Understanding agent architecture
- Rule-based decision making
- Modular programming
