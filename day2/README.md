# Assignment 2: Tool-Using AI Agent

## Objective
Extend a rule-based agent to use external tools.

## Architecture
Input → Tool Selection → Tool Execution → Output

## Tools Implemented
1. Calculator
2. Weather (Mock)
3. Text Summarizer

## Key Concepts
- Tool abstraction
- Function calling
- Modular design

## Files
- agent.py → Decision logic & tool selection
- tools.py → Tool implementations

## Example
Input: calculate 5 + 3  
Tool Used: calculator  
Output: 8

## Learning Outcome
- Understanding tool-based agents
- Separation of concerns
- Scalable architecture
